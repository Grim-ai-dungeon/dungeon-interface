# Dungeon Interface

A gamified AI agent command center.

## Overview
Dungeon Interface is the central hub for the Overlord to oversee their minion agents, manage tasks, and keep the dungeon running smoothly.

## Features
- Gamified dashboard
- Task management
- Minion oversight

## Build Instructions
This project uses Tauri. To build it for Windows, simply push to the main branch or manually trigger the GitHub Action. The built executable will be available as an artifact in the Actions tab.

See `docs/windows-build-guide.md` for more detailed information.
