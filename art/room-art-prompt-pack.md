# Dungeon Interface Room Art Prompt Pack

Goal: create the remaining weakest room art pieces for the dungeon interface with consistent professional indie-game quality.

## Global style anchor
- dark dungeon-tech hybrid
- worn stone, black metal, cyan glow, subtle purple accents
- readable silhouettes, layered depth, low visual noise for UI use
- cinematic but not cluttered
- no text, no logos, no watermark
- consistent perspective and lighting across rooms

## Room 1 — Command Hall / DM Chamber
**Use:** primary background / hero room
**Prompt:**
A grand dungeon command hall for a minion-control interface, central throne-like command dais with monitors and rune-lit consoles, towering stone arches, hanging chains, glowing cyan circuitry embedded in ancient masonry, dark metal trims, subtle purple ambient fog, symmetrical composition, dramatic low-angle perspective, cinematic lighting, high readability, polished indie game environment art, no characters, no text, no watermark, 16:9

**Negative prompt:**
text, letters, watermark, logo, clutter, blurry, low detail, flat lighting, cartoonish, photorealistic humans, busy interface elements

## Suggested iteration order
1. Command Hall
2. Treasury Vault
3. Summoning / Dispatch Chamber
4. Research / Scouting Chamber
5. Forge / Build Chamber

## Notes
- Keep all rooms on the same camera language: slightly elevated, wide composition, strong foreground-to-background layering.
- Keep lighting consistent: cyan key light, purple ambient shadows, selective warm highlights for special rooms.
- Avoid busy props unless they support the room's job.
- If a room feels weak, simplify first before adding detail.
